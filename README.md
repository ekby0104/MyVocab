## ✨ 주요 기능

### 📖 단어 관리
- **전체 단어 / 즐겨찾기 / 검색** 탭 분리
- **6가지 정렬**: 최신순, 알파벳, 알파벳 역순, 즐겨찾기, 오답 순 (isWrong 우선 + wrongCount 내림차순), 랜덤
- **뜻 보이기/숨기기**: 목록에서 뜻을 숨겨서 셀프 테스트 가능
- **스와이프 액션**: 즐겨찾기 토글 + 삭제
- **단어 직접 추가/편집**: 영어, 발음, 품사, 뜻, 예문, 메모 전부 편집 가능
- **단어 상세 화면**: 학습 통계, 메모, TTS, 개별 학습 기록 관리

### 🔍 강력한 검색
- **한/영 통합 검색**: 영어, 한글 뜻, 발음, 예문 모두 검색 대상
- **검색 범위 선택**: 전체 / 단어 / 예문 필터
- **와일드카드 지원**:
  - `ab*` — ab로 시작 (뒤에 1글자 이상)
  - `*ing` — ing로 끝 (앞에 1글자 이상)
  - `c*t` — c와 t 사이에 1글자 이상
  - `c?t` — c+한글자+t (cat, cot, cut)
- **검색 기록**: 최근 10개 자동 저장, 탭으로 재검색, 개별/전체 삭제
- **일괄 즐겨찾기**: 검색 결과 전체를 한 번에 즐겨찾기

### 🎮 학습 모드 (게임 탭에서 시작)
모든 게임은 시작 전에 **대상을 선택**할 수 있습니다:
- 📚 전체 단어
- ⭐ 즐겨찾기
- 🔄 틀린 단어
- 📅 오늘의 학습 (SRS 기반)

#### 🎴 플래시카드
- 앞면(영어) ↔ 뒷면(뜻/예문) 탭으로 전환
- 🔊 수동 TTS 버튼 + 자동 발음 토글
- 섞기, 다시 선택 메뉴

#### 🎯 퀴즈 (4지선다)
- 영→한 / 한→영 모드 선택
- 문제 수 커스텀 (프리셋 10~1000개 + 직접 입력)
- ⏱ **5초 자동 오답**: 시간 내에 답하지 않으면 자동 오답 처리 (찍기 방지)
- SRS 자동 반영 (정답→레벨 업, 오답→레벨 0)

#### 🔀 단어 매칭
- 영어 8장 + 한글 8장 = 16장 카드 4×4 그리드
- ⏱ **30초 제한시간** (타이머 바 + 잔여 시간 표시)
- 같은 종류(영-영, 한-한) 선택 방지
- 매칭 성공 → 초록 깜빡, 실패 → 빨간 깜빡
- 시간 초과 시 미매칭 단어 전부 오답 처리
- SRS 자동 반영

### 📅 간격 반복 학습 (SRS)
Leitner box 방식의 8단계 레벨 시스템:
| 레벨 | 복습 간격 |
|------|----------|
| 0 | 즉시 (오답) |
| 1 | 1일 후 |
| 2 | 3일 후 |
| 3 | 7일 후 |
| 4 | 14일 후 |
| 5 | 30일 후 |
| 6 | 60일 후 |
| 7 | 120일 후 (마스터) |

정답 시 레벨 +1, 오답 시 레벨 0으로 리셋.

### 📊 학습 통계
- **요약 카드**: 전체 단어, 학습 완료, 마스터, 정답률, 즐겨찾기, 틀린 단어
- **학습 누적 바**: 정답/오답 비율 시각화
- **SRS 레벨 분포**: 레벨별 단어 수 막대 그래프
- **자주 틀리는 단어 TOP 10**: 순위와 오답 횟수 표시, 탭하면 상세로 이동

### 🔊 TTS 발음
- `AVSpeechSynthesizer` 기반 영어 발음 (미국식, 학습용 속도 0.45x)
- 단어 상세: 단어 + 예문 각각 발음 버튼
- 플래시카드: 수동 버튼 + 자동 발음 토글
- 퀴즈: 영→한 모드에서 발음 버튼
- 단어 편집: 입력 중 발음 미리듣기

### 🔐 네이버 동기화
- **WebView 기반 로그인**: 네이버 공식 로그인 페이지 사용
- **쿠키 영구 저장**: 한 번 로그인 후 만료 전까지 자동 유지
- **빠른 동기화**: 탭 한 번으로 전체 단어 업데이트
- **동적 단어장 목록**: API에서 단어장 목록 자동 조회 (하드코딩 ID 없음)
- **세션 만료 자동 감지**: 만료 시 재로그인 유도

### 🛡 학습 기록 관리 (3단계)

**단어 상세 (개별)**:
| 버튼 | 동작 |
|------|------|
| 오답 카운트 초기화 | wrongCount만 0 |
| 틀린 단어 해제 | isWrong만 false |
| 학습 기록 초기화 | 정답·오답·SRS·틀린 상태 전부 리셋 |

**게임 탭 → 관리 (전체)**:
| 버튼 | 동작 |
|------|------|
| 틀린 단어 전체 해제 | 모든 단어 isWrong → false |
| 오답 기록 전체 초기화 | 모든 단어 wrongCount → 0 |
| 학습 기록 전체 초기화 | 모든 단어 정답·오답·SRS·틀린 상태 전부 리셋 |

### 💾 백업 / 복원

## 🛠 기술 스택

- **SwiftUI** — 전체 UI (iOS 17+)
- **SwiftData** — 단어 저장소 (`@Model`, `@Query`)
- **WKWebView** — 네이버 로그인
- **URLSession** — 쿠키 기반 API 호출
- **AVSpeechSynthesizer** — TTS
- **Combine** — 타이머 기반 자동 오답 처리
- **UserDefaults / @AppStorage** — 쿠키, 검색 기록, 단어장 목록, 설정 저장

## 🚀 시작하기

### 요구사항
- macOS 14+
- Xcode 15+
- iOS 17+ 실기기 또는 시뮬레이터

### 빌드
1. 이 저장소를 clone
2. `MyVocab.xcodeproj` 열기
3. Signing & Capabilities에서 본인 Apple ID 선택
4. ⌘R로 실행

### 첫 사용 흐름
1. **설정 → 네이버 로그인** → WebView에서 네이버 로그인
2. **단어장 목록 새로고침** → 본인의 단어장 목록 자동 로드
3. **단어장 선택** → 드롭다운에서 원하는 단어장
4. **빠른 동기화** → 단어 가져오기
5. **게임 탭** → 플래시카드 / 퀴즈 / 매칭 선택 → 학습 시작!
